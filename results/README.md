# Example Results

Use this folder for sanitized, repeatable artifacts from live runs.

Recommended contents:

- `large-repo-query.json` - saved GitNexus query output
- `plain-search-comparison.txt` - saved `compare_query.sh` output
- `symbol-context.json` - saved GitNexus context output for the top symbol
- `impact-summary.json` - saved GitNexus impact output

Do not commit private paths, API keys, personal access tokens, conversation
secrets, or raw work logs.

For public examples, keep results concise and inspectable. A reader should be
able to understand the command, the input, and the useful GitNexus output
without needing access to your local machine.
